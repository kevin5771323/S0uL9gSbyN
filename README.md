# 宿舍管理系统简介 java1130

## 项目概述

宿舍管理系统是基于`node.js`与`vue`的前后端分离架构，旨在为高校宿舍管理提供高效、便捷的信息化解决方案。

## 技术栈

- **前端**：Vue.js
- **后端**：Node.js
- **数据库**：MySQL

## 功能模块

系统主要包括以下功能模块：

- 学生信息管理
- 宿舍楼信息管理
- 缺勤与请假管理
- 维修与投诉处理
- 数据统计分析

## 系统角色

系统定义了以下角色：

- 超级管理员
  - 系统全局管理权限
- 管理员
  - 管理所负责的宿舍区域
- 学生
  - 提交请假申请、维修请求等

## 运行环境

- 前端：任何现代浏览器
- 后端：支持Node.js的服务器环境
- 数据库：MySQL数据库服务器

## 部署步骤

1. 安装后端依赖库
2. 配置数据库连接
3. 部署前端代码至Web服务器
4. 启动后端服务

## 目录结构

```
根目录/
├── 前端代码/
│   ├── assets/
│   ├── components/
│   ├── views/
│   ├── router/
│   └── store/
└── 后端代码/
    ├── models/
    ├── routes/
    ├── controllers/
    └── middleware/
```

## 核心亮点

- 前后端分离，便于维护与扩展
- 细致的权限控制，保障系统安全
- 界面简洁，操作便捷
- 开放的API接口，支持与其他系统集成
